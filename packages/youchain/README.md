# youchain

This is the main package of [youchain.js](https://github.com/youchainhq/youchain.js)

## Installation

### Node.js

```bash
npm install youchain
```

### Usage

```js
import YOUChain from 'youchain';

new YOUChain(...);
```

## Types

All the typescript typings are placed in the types folder.
